Метод not_found используется в случае, когда страница не найдена (ошибка 404). Отображает страницу public/404.html

#token_error(exception) ⇒ Object (private)

Note:

Вызывается при получении ошибки ActionController::InvalidAuthenticityToken при попытке отправить некорректные данные из формы

Метод token_error перенаправляет пользователя на предыдущую страницу, которую получает из request.referrer. Дополнительно передает flash сообщение - danger: “Ошибка, повторите отправку формы”
